Aiuto rapido: come funziona l'app

L'app gira in locale e usa un unico motore cloud: Azure Document Translator in modalita sincrona con upload diretto del file (senza Blob Storage).

  • Motore unico: Azure Document Translator per i formati supportati in modalita sync.
  • Nessun Blob richiesto: il file viene inviato direttamente all'endpoint sync.
  • PDF con Azure: disponibili via modalita batch usando Azure Blob Storage.
  • Lingue principali: sorgente PT, target EN/IT.
  • Output: file tradotti in data/processed/<data_ora>/<documento>/.
  • OneDrive: import da cartella locale gia sincronizzata, senza App Registration.

Configura AZURE_TRANSLATOR_ENDPOINT e AZURE_TRANSLATOR_KEY. Per PDF configura anche Blob nella tab Settings.
